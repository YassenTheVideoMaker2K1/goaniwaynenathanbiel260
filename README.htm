If you don’t know. GoAniWayneNathaniel2600 is based on Wrapper Offline version 2.0.0 With Mods applied and Some Wayne Nathaniel OCs.

1.    How to use it

2.    FAQs

 

 

1.   How to use it.

 

 

So This One is portable. Meaning No installer required. But You have to extract it from the ZIP File.

 

The first thing is to do is to extract it from ZIP. Once you already extracted it. You can now go to next step

 

Step 2 :

 

Install the Dependencies. GoAniWayneNathaniel2600 is dependent on Flash and Node.js

 

First open the Install these First folder

 

Next Launch the installer for the adobe flash and Node.js. Once you installed those dependencies you may go to the next step

 

Step 3 execute FIX Local host errors.BAT

 

You need to do this to fix the Local Host Errors

 

 

 

 

 

 

Step 5.

 

Once you fix the Localhost errors. You should execute the NPM AUDITOR.BAT to Fix the npm start . Execute START.BAT next to let the Video Maker open, Once you do this and you are now fine

 

 

You can create characters, Make Videos and whatsoever after it’s installed

 

Note :

If you want to re-open this Program. Execute the START.BAT To Launch GoAniWayneNathaniel2600

 

 

 

 

2.    FAQs

 

 

1.    Does it run on Windows xp ?

 

No Wrapper Offline doesn’t run on XP

 

 

 

2.    What storage size I need ?

 

You need at least 2 gb of free disk storage

 

 

3.    Where are the new assets ?

 

To access it. First select Make a video. After you select it you will get prompted on what theme. Select Comedy world. After selecting Comedy World Click the dropdown Icon from the Ones highlighted with Blue

 

After that. Select Your Library option.

 

Next you will see new backgrounds, new Props and New Sounds

 

Now you get them

 

 

 

4.    Why There are Pre-made Characters ?

 

They are Equipped along with the other new assets

 

 

 

System Requirements :

 

RAM :

At least 1GB of ram

 

Operating System :

You should use Windows 10 1507 at least. But you can use Windows 10 build 9926. It works on that build. Windows 11 is also compatible. But as for Windows Vista, Windows 8 are unsupported due to a Sharp module error related message seen here.

 

Windows 7, Windows 8.1 and Windows Server 2012 R2 is Supported with Node.js Version 11 and hotfixes applied. Read the instructions on how to install node.js 11 and how to apply hotfixes in Windows 8.1 inside of the node.js 11 (If you use Windows 7 or Windows 8.1) folder from the Install These Folder and you will see the Read This First.htm file

 

 

Windows 95, Windows 98 and Windows ME is also not supported because many dependencies including the node.js and flash will refuse to work on these systems because of being a part of The Windows 9x line.

 

Windows NT 3x, Windows NT 4, Windows 2000 and Windows XP is also incompatible due to dependencies will not work.

The Same effect as Windows Server 2003 and Windows Server 2003 R2.

 

As for Windows Server 2012 R2 (With Hotfixes and Node.js 11) Windows Server 2016, Windows Server 2019 and Windows Server 2022 they are working even if untested due to They use the Same Codebase as Windows 8.1, Windows 10 and Windows 11 Respectively

 

Linux and MacOS are Not tested. But chances are that it won’t work on these Operating Systems. As it is Optimized to work in Windows systems.

 

Storage/Free space :

 

2GB. But It is recommended that you use 6GB For your assets. The Archive is 1.26 GB but if you make changes and it will make the Directory grow up

 

Graphics :

 

It is recommended you use 1024x768 Resolution at least. 800x600 is a bit crowded to use the Software. But it still works just it will have problems on displaying the resources. 640x480 is not recommended. It will make items crowded and Cut off on many times.

 

The GPU doesn’t need WDDM Drivers (Aero Compatible)
